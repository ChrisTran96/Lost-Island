# Lost-Island

## Control

### Moving
Arrow keys or  W
              ASD
### Interacting
SPACE bar

### Open pause menu
*In town and house only*
ESC key

### Quit a puzzle
ESC key

### Play logic gate puzzle
Click on a logic gate to move them around and move it to a "box" to finish it.

### Play binary puzzle
To write a number in a "box", click on it.

## Other
"Hint" and "Clear" box can be clicked on.
